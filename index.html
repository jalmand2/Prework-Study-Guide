Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML elements work together to build the structure of a webpage.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • CSS can be applied either inline CSS, internal CSS style sheet or external CSS style sheet.
  • A link element is placed in the HTML's head when using an external CSS style sheet to link the CSS to the HTML file.
  • The CSS rule contains a selector and declarations within that with properties and values.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • cd helps us navigate to a specific location or directory, and cd .. helps us revert from a directory
